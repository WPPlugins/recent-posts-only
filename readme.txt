Restricts posts displayed on the home page to a specified length of time.

== Description ==

Restricts posts displayed on the home page to a specified length of time. Posts displayed via widgets or on any other page (archives, etc.) are unaffected. Options are in Settings > Reading.

== Installation ==

1. Simply upload the recent-posts-only folder to your plugin folder, like any other plugin

2. Enter your WordPress Dashboard, go to Plugins, and activate as usual

3. Now in Settings->Reading there is a new option under "Recent Posts Only"
